üìä Dashboard Quick Tasks

View Donation Report

  1. Go to Dashboard
  2. Find "Donation Report" widget
  3. Select month/year dropdown
  4. View total and breakdown

Check Critical Stock

  1. Dashboard ‚Üí "Critical Stock Report"
  2. Red items = critically low
  3. Yellow items = low stock
  4. Click item to view details

üíä Medicine Stock Tasks

Add New Medicine

  1. Medicine Stock ‚Üí "+ Add Medicine"
  2. Fill name, category, quantity, unit, min level
  3. Click "Add Medicine"
  4. Go to "Manage Batches" to add batch details

Find Low Stock Items

  1. Medicine Stock page
  2. Filter by status dropdown
  3. Select "Low" or "Critical"
  4. View filtered items

Update Stock Quantity

  1. Find medicine in list
  2. Click "Edit"
  3. Change quantity
  4. Click "Save"

üöö Distribution Tasks

Create Distribution to Location

  1. Distribution ‚Üí "+ New Distribution"
  2. Select "By Location"
  3. Choose delivery center
  4. Add medicines and quantities
  5. Click "Create Distribution"

Create Patient Distribution

  1. Distribution ‚Üí "+ New Distribution"
  2. Select "By Patient"
  3. Search for patient
  4. Add medicines
  5. Set urgency level
  6. Click "Distribute"

Track Distribution Status

  1. Distribution ‚Üí View History
  2. Filter by location/patient/date
  3. Check status (Pending/In Transit/Delivered)
  4. Click for detailed view

üí∞ Costing & Reports

Generate Location Cost Report

  1. Costing ‚Üí "Location Cost Report"
  2. Select location
  3. Choose month/year
  4. Review cost breakdown
  5. Export as PDF/Excel

Export Report

  1. Generate report
  2. Click "Export" button
  3. Choose format (PDF/Excel/CSV)
  4. File downloads automatically
