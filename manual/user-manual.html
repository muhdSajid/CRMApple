üíä Medicine Stock Management

Accessing Medicine Stock

  1. Click "Medicine Stock" in the left sidebar
  2. You'll see a list of all medicines with stock levels

Understanding Stock View

Column Description
Medicine Name Name of the medicine
Category Medicine category (Antibiotic, Pain Relief, etc.)
Quantity Current stock quantity
Unit Unit of measurement (tablets, bottles, ml)
Minimum Level Minimum quantity to maintain
Status Stock status (In Stock, Low, Critical)

Stock Status Colors

  • üü¢ Green (In Stock) - Good quantity
  • üü° Yellow (Low Stock) - Close to minimum
  • üî¥ Red (Critical) - Below minimum, urgent reorder

Adding New Medicine

  1. Click "+ Add Medicine" button
  2. Fill in the form:
    • Medicine Name: e.g., "Aspirin 500mg"
    • Category: Select from dropdown
    • Quantity: Current stock amount
    • Unit: tablets, bottles, ml, etc.
    • Minimum Level: Reorder point
    • Manufacturer: Medicine manufacturer
    • Price Per Unit: Cost per unit
  3. Click "Add Medicine"

Managing Batches (Lots)

What is a Batch?

A batch is a group of medicines from the same production run, identified by a unique lot number.

Adding a Batch

  1. Select a medicine from the list
  2. Click "Manage Batches" or "Add Batch"
  3. Fill batch details:
    • Batch Number: Unique identifier (LOT20250001)
    • Expiry Date: When medicine expires
    • Quantity: Number of units
    • Manufacturing Date: When produced
    • Cost Price: Cost per unit
  4. Click "Add Batch"

Search & Filter

Search

  • Type medicine name in search box
  • Results filter in real-time

Filter

  • By Status: All, In Stock, Low, Critical
  • By Category: Select medicine type
  • By Date: Date range filter

üöö Medicine Distribution

Understanding Distribution

Distribution is when medicines are sent to a location or patient.

Distribution Types

1. By Location

Send medicines to delivery centers

  • Select delivery center
  • Add medicines and quantities
  • Specify received by person
  • Add special notes

2. By Patient

Give medicines to individual patients

  • Search and select patient
  • Set urgency level (Normal, Urgent, Critical)
  • Add medicines with dosage
  • Add delivery instructions

Distribution Status

Status Meaning
üü¢ Pending Ready to process
üü° In Transit On the way
üîµ Delivered Successfully received
‚ö´ Cancelled Not going
üî¥ Failed Delivery problem

How to Create a Distribution

Distribution by Location

  1. Click "Medicine Distribution" in sidebar
  2. Click "+ New Distribution" ‚Üí "By Location"
  3. Select delivery center
  4. Enter distribution date
  5. Enter received by (person name)
  6. Click "Add Medicine" for each medicine:
    • Select medicine
    • Enter quantity
    • Confirm unit price
  7. Review total cost
  8. Click "Create Distribution"

Distribution by Patient

  1. Click "Medicine Distribution" in sidebar
  2. Click "+ New Distribution" ‚Üí "By Patient"
  3. Search for patient by name/ID
  4. Set delivery date
  5. Set urgency level
  6. Add medicines:
    • Select medicine
    • Enter prescribed quantity
    • Add notes (dosage instructions)
  7. Click "Distribute"

Viewing Distribution History

  1. Go to Medicine Distribution
  2. Click "View History"
  3. Filter by:
    • Date range
    • Location
    • Patient name
    • Status
  4. Click on a distribution to view details
